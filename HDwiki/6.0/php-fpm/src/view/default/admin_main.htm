第一步：修改站点基本信息
站点基本信息，是你网站区别于其他站点的标识，搜索引擎收入的重要依据。
操作步骤：
1：全局-->站点设置,修改站点名称，网站url，站点公告等。

1/5 下次登录时显示

第二步：设置分类，创建词条
词条是HDWiki对外展示的主要内容，分类则让词条的展示更加明晰。
操作步骤：
1：内容管理--> 分类词条管理；创建您网站的分类
2：创建词条—> 创建新的词条，导入网站的相应分类。
3：内容管理--> 词条管理，分别设置热门词条，精彩词条，推荐词条

2/5 下次登录时显示

第三步：优化站点负载，提升用户体验
站点的负载，通俗上是指一个站点同一时间，所能承受的最大访问量。通常情况下，高访问量势必带来低访问效果，从而降低用户体验。通过以下设置，可在不改动硬件环境的情况下，优化站点负载，提升用户体验，降低运营风险。
操作步骤：
1：百科联盟--> 云搜索；设置开启云搜索，提升搜索体验，降低空间的搜索压力
2：数据库管理--> 数据库存储设置；设置历史版本，存储为文本格式。减少数据库消耗

3/5 下次登录时显示

第四步:定期推送精彩内容，提高网站回访率
HDWiki开启了独特的分享模式，可以让用户自由分享到开心，人人，新浪微博等平台，通过定期推送精彩内容，可以极大地提升网站回访率。
操作步骤：
1：创建词条--> 选择分类创建新的词条。
2：内容管理-->设置热门词条，精彩词条，推荐词条。
3：手动分享-->在词条页面点选按钮分享至其他平台。

4/5 下次登录时显示

第五步：使用自定义菜单，设置自己常用的菜单
HDWiki提供了自定义菜单功能，站长可以设置，自己经常使用的功能菜单。最多设置5个，不能贪多哦。
操作步骤：
1：在后台的右上角，logo下，找到自定义菜单。
2：点击右边的设置，弹出设置快捷功能层。
3：勾选自己常用的功能。
4：点击设置按钮，确认修改。
